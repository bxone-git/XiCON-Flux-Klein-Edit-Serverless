# 앱 > 영상 컨텐츠 기능명세서

## 페이지 개요

- **목적**: 사용자가 선택한 템플릿과 입력 정보를 바탕으로 AI 마케팅 영상 컨텐츠 생성
- **구조**: 좌측 템플릿 선택 패널 + 우측 입력 폼 패널 + 하단 작업 내역 패널
- **접근 권한**: 로그인 필수, 프로젝트 선택 필수

---

## 템플릿 선택 (좌측 패널)

### 기능

- 생성할 영상의 연출 스타일과 효과를 결정하는 템플릿 리스트 제공 및 선택

### 동작

- **템플릿 리스트**: 규격화된 영상 스타일이 카드로 표시되며 슬라이드 형태로 나열
- **템플릿 선택**: 카드 클릭 시 해당 템플릿이 선택 상태로 강조되며 우측 입력 폼의 필드가 해당 템플릿에 맞춰 최적화됨
- **자동 선택**: `앱 > 홈(갤러리)` 또는 `홈페이지 > 공개 갤러리`에서 특정 생성물의 '바로 만들기'를 통해 진입한 경우, 해당 생성물에 사용된 템플릿이 자동으로 선택됨

---

## 영상 생성 폼 (우측 패널)

### 기능

- 선택된 템플릿에 적용할 이미지, 문구 및 설정 정보 입력

### 동작

#### 입력 필드 상세

- **제품/상품 이미지**
  - **컴포넌트 종류**: File Uploader
  - **제약 사항 / 비고**: 필수, JPG/PNG (최대 10MB)
- **영상 홍보 문구**
  - **컴포넌트 종류**: Textarea
  - **제약 사항 / 비고**: 필수, 영상 내 삽입될 메인 카피
- **브랜딩 리포트 선택**
  - **컴포넌트 종류**: Select (Dropdown)
  - **제약 사항 / 비고**: 선택, 현재 프로젝트의 완료된 리포트 연동
- **출력 영상 수**
  - **컴포넌트 종류**: Number Input
  - **제약 사항 / 비고**: 필수

- **생성하기 버튼**
  - 버튼 내에 출력 영상 수에 따라 계산된 소모될 크레딧 표시
  - 모든 필수 정보 입력 및 템플릿 선택 시 활성화
  - 클릭 시 이미지 생성 프로세스 시작
  - 크레딧 차감
  - 하단 작업 내역 패널에 로딩 카드 추가

### 상태별 처리

- **템플릿 미선택**: 템플릿 선택 전까지 입력 폼 비활성화 또는 안내 표시
- **필수 항목 미입력**: '생성하기' 버튼 비활성화
- **크레딧 부족**: '생성하기' 버튼 비활성화 및 설정 모달(충전) 유도

## 작업 내역 및 결과물 (하단 패널)

### 기능

- 현재 프로젝트에서 생성한 영상 컨텐츠 리스트 확인 및 관리
- 생성 상태 실시간 확인 및 프리뷰

### 리스트 및 카드 동작

- **카드 나열**: 생성된 결과물들이 카드 형태로 슬라이드 나열
- **호버 재생**: 결과물 카드에 마우스 호버 시 영상이 자동 프리뷰 재생됨
- **로딩 카드**: 생성 시작 시 진행 상태 및 예상 소요 시간 표시
- **결과물 카드**: 생성 완료 시 영상 썸네일 표시
- **상세 보기**: 카드 클릭 시 전체화면 다이얼로그로 영상 재생 및 상세 정보 표시
  - 상세 보기 UI/정책(좋아요, 만족도, 공개 설정 등)은 공통 컴포넌트 명세를 따름: [생성물 상세보기 모달](../common/gallery-detail-modal.md)
- **피드백(만족도)**: **좋아요/싫어요 중 1개를 단 1회** 피드백 가능 (변경/취소 불가)
- **재시도**: 생성 실패 시 해당 카드에 재시도 버튼 노출

### 상태별 처리

- **생성 완료**: 썸네일 표시 및 호버 시 재생, 상세 보기 활성화
- **생성 실패**: 에러 아이콘 및 재시도 버튼 표시
- **내역 없음**: "생성된 영상 컨텐츠가 없습니다" 안내 문구 표시

---

## 공통 동작

- **프로젝트 연동**: 현재 선택된 프로젝트의 데이터만 노출
- **브랜딩 리포트 활용**: 리포트 선택 시 브랜드 정체성을 반영한 영상 연출 및 톤앤매너 적용
- **상세 보기 액션**: 상세 보기 내에서 '다운로드', '즐겨찾기 추가(즐겨찾기)' 등의 기능 제공
