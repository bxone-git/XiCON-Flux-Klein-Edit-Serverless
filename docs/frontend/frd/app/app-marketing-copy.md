# 앱 > 마케팅 문구 기능명세서

## 페이지 개요

- **목적**: 마케팅 문구 또는 SNS 게시물용 텍스트 컨텐츠 생성
- **구조**: 좌측 입력 폼 패널(탭 구성) + 우측 작업 내역 패널(그리드)
- **접근 권한**: 로그인 필수, 프로젝트 선택 필수, 생성된 브랜드 리포트 보유 필수

---

## 마케팅 문구 생성 폼 (좌측 패널)

### 기능

- 목적에 따른 탭 구분(마케팅 문구 / SNS 게시물) 및 정보 입력
- 브랜딩 리포트 연동을 통한 최적화된 문구 생성

### 동작 (탭별 입력 항목)

#### 입력 필드 상세

- **마케팅 문구**
  - **제품 한 줄 소개**
    - **컴포넌트 종류**: Input (Text)
    - **제약 사항 / 비고**: 필수
  - **강조 메시지**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수
  - **최대 글자 수**
    - **컴포넌트 종류**: Number Input
    - **제약 사항 / 비고**: 필수
- **SNS 게시물**
  - **SNS 종류**
    - **컴포넌트 종류**: Select
    - **제약 사항 / 비고**: 필수, 인스타그램/페이스북/블로그 등
  - **컨텐츠 내용**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수, 최소 10자 이상
  - **컨텐츠 언어**
    - **컴포넌트 종류**: Radio/Toggle
    - **제약 사항 / 비고**: 필수, 한국어/영어
  - **추가 요청사항**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 선택, 톤앤매너 등 자유 서술
- **공통**
  - **브랜딩 리포트 선택**
    - **컴포넌트 종류**: Select (Dropdown)
    - **제약 사항 / 비고**: 필수, 현재 프로젝트의 완료된 리포트 리스트

- **생성하기 버튼**
  - 클릭 시 문구 생성 프로세스 시작
  - 크레딧 차감
  - 작업 내역 패널에 로딩 카드 추가

### 상태별 처리

- **리포트 미선택**: 리포트 선택 전까지 '생성하기' 버튼 비활성화
- **입력값 미달**: 필수 항목(소개 또는 내용) 미입력 시 버튼 비활성화

---

## 작업 내역 및 결과물 (우측 패널)

### 기능

- 생성된 문구 내역 확인 및 관리
- 결과물 다운로드 및 사용자 피드백

### 리스트 및 카드 동작

- **로딩 카드**: 생성 중 상태 표시 (예상 시간 등)
- **결과물 카드**: 생성 완료 시 문구 요약 썸네일 표시
- **상세 보기/다운로드**: 카드 클릭 시 전체화면으로 상세 문구 확인 및 결과물(PDF) 다운로드
- **피드백(만족도)**: **좋아요/싫어요 중 1개를 단 1회** 피드백 가능 (변경/취소 불가)
- **재시도**: 생성 실패 시 재시도 버튼 노출

### 상태별 처리

- **생성 완료**: 결과물 요약 표시 및 액션 버튼 활성화
- **생성 실패**: 에러 아이콘 및 재시도 버튼 노출
- **내역 없음**: "생성된 마케팅 문구가 없습니다" 안내 문구 표시

---

## 공통 동작

- **프로젝트 연동**: 현재 선택된 프로젝트의 데이터만 노출
- **브랜딩 리포트 의존성**: 마케팅 문구 생성은 반드시 기존에 생성된 브랜딩 리포트가 있어야 가능함 (없을 시 리포트 생성 페이지로 이동)
- **크레딧 체크**: 생성 시도 시 보유 크레딧 확인 및 부족 시 충전을 위해 설정 모달 열기
