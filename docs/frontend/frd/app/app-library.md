# 앱 > 라이브러리 기능명세서

## 페이지 개요

- **목적**: 현재 프로젝트 내에서 생성된 모든 종류의 컨텐츠(리포트, 문구, 이미지, 영상)를 통합 관리
- **구조**: 상단 필터 및 제어바 + 메인 컨텐츠 그리드 영역
- **접근 권한**: 로그인 필수, 프로젝트 선택 필수

---

## 필터 및 제어 (상단 헤더)

### 기능

- 컨텐츠 종류별 필터링, 즐겨찾기 필터링 및 일괄 편집 모드 제어

### 동작

- **종류별 필터**: 브랜딩 리포트, 마케팅 문구, 홍보 이미지, 영상 컨텐츠 중 선택하여 필터링 (다중 선택 가능)
- **즐겨찾기 필터**: 별표 아이콘 클릭 시 즐겨찾기(즐겨찾기 추가)한 컨텐츠만 노출
- **선택 모드**: '선택' 버튼 클릭 시 일괄 편집 모드로 전환
  - **전체 선택**: 현재 필터링된 모든 컨텐츠 일괄 선택
  - **선택 취소**: 일괄 편집 모드 종료
  - **일괄 즐겨찾기**: 선택된 컨텐츠들을 일괄적으로 즐겨찾기 추가/해제
  - **일괄 삭제**: 선택된 컨텐츠들을 일괄적으로 휴지통으로 이동

---

## 컨텐츠 리스트 (그리드 영역)

### 기능

- 필터링된 컨텐츠들을 카드 형태로 표시하고 개별 액션 제공

### 동작

- **카드 구성**: 컨텐츠 종류별 아이콘, 썸네일(이미지/영상), 제목(브랜드명/제품명), 생성일 표시
- **개별 액션**:
  - **카드 클릭**: 전체화면 상세 보기 다이얼로그 노출 (상세 정보 확인 및 다운로드)
    - 상세 보기 UI/정책(좋아요, 만족도, 공개 설정 등)은 공통 컴포넌트 명세를 따름: [생성물 상세보기 모달](../common/gallery-detail-modal.md)
  - **즐겨찾기 버튼**: 카드 상단의 별 버튼으로 즐겨찾기 토글
- **선택 모드 시**: 카드 클릭 시 상세 보기 대신 체크박스 선택/해제 동작 수행

### 상태별 처리

- **로딩 중**: 카드 그리드 영역 스켈레톤 표시
- **결과물 없음**: "라이브러리가 비어 있습니다. 새로운 컨텐츠를 생성해 보세요!" 안내 및 생성 페이지 바로가기 링크 표시

---

## 공통 동작

- **프로젝트 연동**: 헤더에서 선택된 현재 프로젝트의 생성물만 표시
- **휴지통 이동**: 삭제된 컨텐츠는 즉시 삭제되지 않고 '휴지통' 메뉴로 이동됨
- **실시간 반영**: 필터 변경 시 새로고침 없이 즉시 그리드 영역 데이터 갱신
