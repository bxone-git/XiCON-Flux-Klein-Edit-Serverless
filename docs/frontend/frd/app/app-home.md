# 앱 > 홈 기능명세서

## 페이지 개요

- **목적**: 주요 생성 기능 빠른 접근 + 다른 사용자들의 공개 생성물 탐색
- **접근 권한**: 로그인 필수

---

## 프로모션 배너

### 기능

- 이벤트/프로모션 홍보용 배너
- 복수 배너 시 캐러셀로 자동 롤링 (5초 간격, hover 시 일시정지)

### 동작

- 배너 클릭 → 지정된 링크로 이동

### 상태별 처리

- **로딩 중**: 스켈레톤
- **등록된 배너 없음**: 섹션 숨김
- **로드 실패**: 섹션 숨김

---

## 새 기능 배너

### 기능

- 신규 기능 안내용 배너
- 닫기 버튼으로 숨길 수 있음

### 동작

- 닫기 클릭 → 배너 숨김, 다음 방문 시에도 숨김 유지 (계정 기준)
- 자세히 보기 클릭 → 해당 기능 안내 페이지로 이동

### 상태별 처리

- **이미 닫음**: 표시 안 함
- **등록된 배너 없음**: 섹션 숨김

---

## 빠른 생성 버튼 그룹

### 기능

- 4개 생성 메뉴 바로가기: 리포트 생성 / 카피 생성 / 이미지 생성 / 동영상 생성
- 클릭 시 해당 생성 페이지로 이동

### 동작

- 버튼 클릭 → 해당 생성 페이지로 이동
- 이동 시 현재 선택된 프로젝트 컨텍스트를 기준으로 동작

### 상태별 처리

- **프로젝트 미선택**: 버튼 비활성화 + "프로젝트를 먼저 선택하세요" 안내

---

## 앱 내 갤러리

### 기능

- 사용자들이 공개 설정한 생성물(이미지, 동영상)을 탐색
- 실제 생성 작업 시 참고용
- 대상 범위: 전체 사용자 중 공개 설정된 생성물

### 필터/정렬

- **즐겨찾기**: ON/OFF (내가 즐겨찾기한 것만)
- **종류**: 전체 / 이미지 / 동영상
- **검색**: 제품명 또는 문구로 검색
- **정렬**: 좋아요순 / 최신순

### 카드 인터랙션

- **좋아요(타인 생성물)**: 토글. 갤러리에서 타인 생성물에 좋아요/좋아요 취소
- **만족도(내 생성물)**: 내 생성물일 경우 좋아요/싫어요 중 1개를 단 1회 선택(변경/취소 불가)
- **즐겨찾기**: 토글. 해당 생성물의 템플릿을 즐겨찾기 추가/제거
- **바로 만들기**: 해당 템플릿이 선택된 상태로 홍보 이미지 또는 영상 컨텐츠 페이지로 이동
- **자세히 보기**: 전체화면 상세 보기 열림
- **동영상 hover**: 자동 재생 (음소거)

### 페이지네이션

- 무한스크롤 방식

### 상태별 처리

- **로딩 중**: 스켈레톤 카드
- **검색 결과 없음**: 빈 상태 안내
- **로드 실패**: 에러 메시지 + 재시도 버튼
- **모두 로드됨**: "모든 컨텐츠를 불러왔습니다" 안내
