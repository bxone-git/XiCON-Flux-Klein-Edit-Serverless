# 앱 > 휴지통 기능명세서

## 페이지 개요

- **목적**: 현재 프로젝트에서 삭제된 컨텐츠들을 관리하고 복원하거나 영구 삭제
- **구조**: 상단 제어바 + 메인 컨텐츠 그리드 영역
- **접근 권한**: 로그인 필수, 프로젝트 선택 필수

---

## 삭제물 관리 (상단 헤더)

### 기능

- 휴지통 내 컨텐츠에 대한 일괄 복원 및 영구 삭제 제어

### 동작

- **전체 선택**: 휴지통 내 모든 컨텐츠를 일괄 선택
- **복원**: 선택된 컨텐츠들을 원래 위치(라이브러리 등)로 복구하고 휴지통에서 제거
- **영구 삭제**: 선택된 컨텐츠들을 데이터베이스에서 완전히 삭제 (복구 불가능 알림 표시)

---

## 삭제물 리스트 (그리드 영역)

### 기능

- 휴지통에 보관된 컨텐츠들을 카드 형태로 표시하며, 기본적으로 선택 가능한 상태로 노출

### 동작

- **카드 구성**: 라이브러리 카드와 동일하게 구성되나, 삭제된 날짜 정보 추가 표시 가능
- **개별 선택**: 각 카드를 클릭하거나 체크박스를 통해 선택/해제
- **상세 보기 제한**: 휴지통 상태에서는 상세 보기나 다운로드 기능이 제한됨 (복원 후 가능)

### 상태별 처리

- **로딩 중**: 카드 그리드 영역 스켈레톤 표시
- **휴지통 비어 있음**: "휴지통이 비어 있습니다" 안내 문구 표시

---

## 공통 동작

- **프로젝트 연동**: 현재 선택된 프로젝트에서 삭제된 내역만 노출
- **자동 삭제**: (정책에 따라) 삭제 후 30일 경과 시 자동으로 영구 삭제 처리될 수 있음
- **영구 삭제 확인**: 영구 삭제 버튼 클릭 시 "정말 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다" 확인 다이얼로그 노출
