# 앱 사이드바 기능명세서

## 컴포넌트 개요

- **목적**: 앱의 주요 기능 간 이동을 위한 네비게이션 제공
- **사용 위치**: 앱 영역 좌측에 고정
- **접근 권한**: 로그인 필수

---

## 메뉴 구성 및 이동

### 기능

- 앱의 주요 서비스 메뉴 리스트 제공 및 페이지 전환
- 현재 활성화된 메뉴 표시

### 동작

- **메뉴 항목 클릭**: 우측 컨텐츠 영역에 해당 페이지 표시 및 URL 변경
- **활성화 상태**: 현재 접속 중인 페이지의 메뉴 항목을 시각적으로 강조(예: 배경색 또는 텍스트 굵기 변경 등)

### 메뉴 리스트

- **홈**
  - **연결 페이지**: `앱 > 홈`
  - **비고**: 서비스 대시보드 및 앱 내 갤러리
- **브랜딩 리포트**
  - **연결 페이지**: `앱 > 브랜딩 리포트`
  - **비고**: 브랜드 분석 및 리포트 생성
- **마케팅 문구**
  - **연결 페이지**: `앱 > 마케팅 문구`
  - **비고**: AI 카피라이팅 생성
- **홍보 이미지**
  - **연결 페이지**: `앱 > 홍보 이미지`
  - **비고**: 템플릿 기반 이미지 생성
- **영상 컨텐츠**
  - **연결 페이지**: `앱 > 영상 컨텐츠`
  - **비고**: 템플릿 기반 동영상 생성
- **라이브러리**
  - **연결 페이지**: `앱 > 라이브러리`
  - **비고**: 현재 프로젝트의 전체 생성물 확인
- **휴지통**
  - **연결 페이지**: `앱 > 휴지통`
  - **비고**: 삭제된 생성물 보관 및 복원

## 사이드바 제어

### 기능

- 화면 크기에 따른 사이드바 노출 제어 (응답형 대응)
- 사이드바 접기/펼치기 (앱 헤더에 접기/펼치기 버튼 있음)

### 동작

- **데스크탑**: 좌측 상시 고정 노출
- **모바일/태블릿**: 햄버거 메뉴 등을 통해 드로워(Drawer) 형태로 노출 제어

### 상태별 처리

- **메뉴 로딩 중**: 메뉴 리스트 영역 스켈레톤 또는 로딩 표시
- **권한 없는 메뉴**: 메뉴 리스트에서 숨김 또는 비활성화 처리

---

## 공통 동작

- 메뉴 클릭 시 즉시 페이지 이동을 수행하며, 진행 중인 작업이 있는 경우(예: 폼 입력 중) 이탈 확인 다이얼로그를 띄울 수 있음
