# 앱 설정 모달 기능명세서

## 컴포넌트 개요

- **목적**: 구독 관리, 크레딧 내역 확인, 결제 수단 관리 및 회원 정보 수정을 위한 통합 설정 제공
- **사용 위치**: 앱 헤더의 사용자 프로필 메뉴 또는 설정 버튼을 통해 호출 (대형 모달)
- **접근 권한**: 로그인 필수

---

## 탭 구성 및 네비게이션

### 기능

- 설정 항목별로 구분된 탭(구독관리, 크레딧내역, 결제관리, 회원정보관리) 제공
- 각 탭 클릭 시 해당 설정 화면으로 전환

### 동작

- **탭 클릭**: 해당 카테고리의 설정 UI로 전환
- **초기 진입**: 설정 열기 시 기본적으로 '구독관리' 탭 표시

---

## 구독관리

### 기능

- 현재 사용 중인 요금제 및 크레딧 상태 확인
- 요금제 변경(구독) 및 크레딧 개별 충전

### 동작

- **요금제 탭**: 요금제 카드 선택 후 결제 진행
- **크레딧 탭**: 충전할 크레딧 카드 선택 후 결제 진행
- **구독 취소**: 구독 중인 경우 취소 절차 진행

### 상태별 처리

- **무료 사용자**: 현재 요금제를 '무료'로 표시 및 업그레이드 유도
- **구독 중**: 다음 결제일 표시

---

## 크레딧내역

### 기능

- 현재 보유한 크레딧의 종류별(정기, 충전) 잔액 확인 및 사용/충전 이력 조회

### 항목 표시

- **보유 정보**: 현재 총 크레딧, 정기 크레딧 잔액, 충전 크레딧 잔액
- **내역 리스트**: 월별 크레딧 사용 및 충전 일시, 상세 내용, 증감액

---

## 결제관리

### 기능

- 결제 수단(카드 등) 등록/삭제 및 결제 내역 확인

### 동작

- **결제 수단 추가/삭제**: 새로운 카드 등록 또는 기존 카드 삭제
- **결제 내역**: 과거 결제 일시, 상품명, 결제 금액, 영수증 확인

---

## 회원정보관리

### 기능

- 계정 정보(아바타, 닉네임, 이메일 등) 수정 및 알림 설정, 계정 삭제

### 동작

- **계정 정보 수정**: 아바타 사진 변경, 닉네임 변경, 이름 확인, 소셜 로그인 연동 정보(제공자, 제공자 계정 ID) 확인
- **알림 설정**: 광고성 정보 수신 동의(이메일, SMS) 토글 제어
- **계정 삭제**: 탈퇴 절차 진행 및 최종 확인

---

## 공통 동작

### 모달 제어

- **닫기**: X 버튼 클릭 또는 모달 외부 영역 클릭 시 닫힘
- **데이터 저장**: 정보 수정 시 '저장' 또는 '적용' 버튼 클릭 시 서버에 반영
