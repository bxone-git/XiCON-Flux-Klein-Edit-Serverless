# 앱 헤더 기능명세서

## 컴포넌트 개요

- **목적**: 현재 프로젝트 선택 및 관리, 알림 확인, 사용자 정보 및 크레딧 상태 확인
- **사용 위치**: 앱 영역 상단에 고정
- **접근 권한**: 로그인 필수

---

## 앱 사이드바 제어

### 기능

- 앱 사이드바 접기/펼치기

### 동작

- **앱 사이드바 접기/펼치기 버튼 클릭**: 앱 사이드바 접기/펼치기

## 프로젝트 선택

### 기능

- 현재 작업 중인 프로젝트를 표시하고 다른 프로젝트로 전환
- 프로젝트 생성 및 수정 기능 포함

### 동작

- **프로젝트명 클릭**: 프로젝트 리스트 드롭다운 표시
- **리스트에서 프로젝트 선택**: 해당 프로젝트로 전환 및 데이터 갱신
- **새 프로젝트 생성 클릭**: 프로젝트명을 입력하는 '새 프로젝트 생성' 모달 표시
- **프로젝트 수정 클릭**: 현재 프로젝트명을 수정 또는 프로젝트를 삭제하는 '프로젝트 수정/삭제' 모달 표시
- **프로젝트 자동 선택**: 앱 진입시 최근 사용한 프로젝트 자동 선택
- **프로젝트 자동 생성**: 앱 진입시 프로젝트가 하나도 없으면 자동 생성 후 자동 선택

### 상태별 처리

- **프로젝트 미선택**: "프로젝트를 선택하세요" 문구 표시(자동 선택하므로 실제로는 표시되지 않음)
- **리스트 로딩 중**: 드롭다운 내 로딩 스피너 표시
- **프로젝트 1개 미만**: '새 프로젝트 생성' 버튼만 활성화(자동 생성하므로 실제로는 표시되지 않음)

## 알림

### 기능

- 컨텐츠 생성 상태(시작, 완성, 실패 등) 및 주요 시스템 알림 제공

### 동작

- **종 아이콘 클릭**: 최근 알림 리스트 드롭다운 표시
- **알림 항목 클릭**: 관련 상세 페이지로 이동 또는 확인 처리

### 상태별 처리

- **새로운 알림 있음**: 아이콘에 레드 닷(Red Dot) 또는 숫자 배지 표시
- **알림 없음**: "새로운 알림이 없습니다" 안내 문구 표시

---

## 사용자 프로필 및 크레딧

### 기능

- 사용자 아바타 및 현재 보유 크레딧 정보 표시
- 계정 관리 및 로그아웃 접근점 제공

### 동작

- **아바타/크레딧 영역 클릭**: 사용자 메뉴 드롭다운 표시
- **메뉴 항목 구성**:
  - 사용자 정보 (아바타, 사용자명, 이메일)
  - 사용 요금제 및 보유 크레딧 정보
  - **문의하기**: 고객 센터 또는 문의 채널로 연결
  - **설정**: 앱 설정 모달 열기
  - **로그아웃**: 세션 종료 및 로그인 페이지로 이동

### 상태별 처리

- **크레딧 부족**: 보유 크레딧 강조 표시 (예: 빨간색 텍스트)
- **프로필 이미지 없음**: 기본 아바타 이미지 표시

## 공통 동작

### 드롭다운 제어

- 외부 영역 클릭 시 열려 있는 드롭다운 닫힘
- 다른 메뉴 클릭 시 기존 드롭다운 닫히고 새 드롭다운 열림
