# 앱 > 브랜딩 리포트 기능명세서

## 페이지 개요

- **목적**: 브랜드의 핵심 정보와 마케팅 전략을 분석하여 종합적인 브랜딩 리포트(PDF) 생성
- **구조**: 좌측 입력 폼 패널(3단계 스텝) + 우측 작업 내역 패널(그리드)
- **접근 권한**: 로그인 필수, 프로젝트 선택 필수

---

## 브랜딩 리포트 생성 폼 (좌측 패널)

### 기능

- 브랜딩 리포트 생성을 위한 브랜드 및 마케팅 상세 정보 입력
- 3단계 스테퍼를 통한 단계별 정보 수집

### 동작

#### 단계별 입력 필드 상세

- **1단계**
  - **브랜드명**
    - **컴포넌트 종류**: Input (Text)
    - **제약 사항 / 비고**: 필수
  - **업종/카테고리**
    - **컴포넌트 종류**: Input (Text)
    - **제약 사항 / 비고**: 필수
  - **브랜드 성격**
    - **컴포넌트 종류**: Checkbox (직접 입력 가능)
    - **제약 사항 / 비고**: 선택
  - **목표 고객**
    - **컴포넌트 종류**: Checkbox (직접 입력 가능)
    - **제약 사항 / 비고**: 선택
- **2단계**
  - **대표 상품/서비스명**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수
  - **핵심 특징**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수
  - **경쟁 브랜드/상품**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수
- **3단계**
  - **주요 마케팅 목표**
    - **컴포넌트 종류**: Textarea
    - **제약 사항 / 비고**: 필수
  - **문서 페이지 수**
    - **컴포넌트 종류**: Input (Number)
    - **제약 사항 / 비고**: 필수, 최대 5페이지
  - **참고 URL**
    - **컴포넌트 종류**: Input (Text)
    - **제약 사항 / 비고**: 선택
  - **참고 자료**
    - **컴포넌트 종류**: File Uploader (PDF/TXT/JPG/PNG)
    - **제약 사항 / 비고**: 선택

- **생성하기 버튼**
  - 모든 필수 정보 입력 시 활성화
  - 클릭 시 리포트 생성 프로세스 시작
  - 작업 내역 패널에 로딩 카드 추가
  - 크레딧 차감

### 상태별 처리

- **입력 중**: 단계별 필수 항목 미입력 시 '다음' 버튼 비활성화
- **생성 중**: '생성하기' 버튼 로딩 상태로 변경 및 중복 클릭 방지

## 작업 내역 및 결과물 (우측 패널)

### 기능

- 현재 프로젝트에서 생성한 브랜딩 리포트 리스트 확인 및 관리
- 생성 상태(대기, 진행, 완료, 실패) 실시간 확인

### 리스트 및 카드 동작

- **로딩 카드**: 생성 시작 시 예상 소요 시간과 함께 로딩 상태 표시
- **결과물 카드**: 생성 완료 시 리포트 썸네일로 표시
- **다운로드**: 완료된 리포트 카드 클릭 또는 다운로드 버튼을 통해 PDF 파일 다운로드
- **피드백(만족도)**: 생성된 결과물에 대해 **좋아요/싫어요 중 1개를 단 1회** 피드백 가능 (변경/취소 불가)
- **재시도**: 생성 실패 시 해당 카드에 재시도 버튼 노출

### 상태별 처리

- **생성 완료**: 썸네일 및 다운로드 버튼 활성화
- **생성 실패**: 실패 아이콘 및 재시도 버튼 표시
- **내역 없음**: "생성된 브랜딩 리포트가 없습니다" 안내 문구 표시

---

## 공통 동작

- **프로젝트 연동**: 현재 헤더에서 선택된 프로젝트에 속한 내역만 필터링하여 표시
- **크레딧 체크**: 생성 시작 전 보유 크레딧 확인 및 부족 시 충전을 위해 설정 모달 열기
