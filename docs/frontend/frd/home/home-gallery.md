# 홈페이지 > 갤러리 기능명세서

## 페이지 개요

- **목적**: 게스트에게 서비스의 고품질 생성물(이미지, 영상)을 노출하여 서비스 가치를 체감시키고 가입을 유도(CTA)
- **구조**: 메인 컨텐츠 그리드(무한 스크롤)
- **접근 권한**: 전체 (비로그인 상태에서도 탐색 가능, 액션 시 로그인 필요)

---

## 갤러리 그리드 (메인 영역)

### 기능

- 공개 설정된 생성물들을 카드 형태로 노출

### 동작

- **무한 스크롤**: 페이지 하단 도달 시 다음 생성물 리스트를 자동으로 로드
- **카드 구성**: 생성물 썸네일, 우측 하단에 좋아요 수 표시
- **호버(Hover) 액션**:
  - **자세히 보기**: 클릭 시 상세 보기 다이얼로그 노출 (로그인 없이 가능)
  - **바로 만들기**: 클릭 시 해당 템플릿 정보를 가지고 `앱 > 홍보 이미지` 또는 `앱 > 영상 컨텐츠`로 이동 (로그인 필요, 미로그인 시 인증 모달 표시)

---

## 상세 보기 (다이얼로그)

### 기능

- 선택한 생성물의 확대 이미지/영상 및 상세 생성 정보 확인

### 동작

- 상세 보기 UI/정책(좋아요, 만족도, 공개 설정, 다운로드 등)은 공통 컴포넌트 명세를 따름:
  - [생성물 상세보기 모달](../common/gallery-detail-modal.md)
- 본 페이지 특이사항: 비로그인 상태에서도 “자세히 보기” 열람은 가능하나, 참여형 액션은 로그인 필요

---

## 공통 동작

- **로그인 유도(CTA)**: '바로 만들기', '즐겨찾기', '다운로드', '좋아요' 등 모든 참여형 액션 클릭 시 '인증 모달(로그인)'을 자동으로 팝업
- **응답형 레이아웃**: 화면 너비에 따라 그리드의 열(Column) 수가 유연하게 조절됨
