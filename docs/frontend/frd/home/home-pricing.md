# 홈페이지 > 구독/가격 기능명세서

## 페이지 개요

- **목적**: 서비스 요금제 체계와 크레딧 구매 정보를 안내하고 구독을 유도
- **구조**: 요금제 선택 섹션 + 크레딧 충전 섹션 + FAQ 섹션
- **접근 권한**: 전체 (비로그인 상태에서 정보 확인 가능, 구매 시 로그인 필요)

---

## 프로모션 배너 (상단)

### 기능

- 요금/구독 관련 이벤트 및 공지사항을 강조 노출 (예: 연간 결제 할인, 신규 가입 혜택, 한정 프로모션)
- 필요 시 복수 배너를 캐러셀로 노출

### 동작

- **노출 위치**: 페이지 최상단 (헤더 아래, 본문 시작 전)
- **배너 클릭**: 지정된 링크(내부/외부)로 이동
- **닫기(선택)**: 닫기 버튼이 있는 배너는 닫을 수 있으며, 닫은 배너는 다음 방문 시에도 숨김 유지 (계정 기준 또는 브라우저 기준 정책 선택)

### 상태별 처리

- **로딩 중**: 스켈레톤 또는 로딩 표시
- **등록된 배너 없음**: 섹션 숨김
- **로드 실패**: 섹션 숨김 또는 기본 안내 문구 표시

---

## 요금제 선택 (Subscription Plans)

### 기능

- 월간/연간 구독 서비스의 종류와 혜택 비교 및 선택

### 동작

- **플랜 카드**: 무료(Free), 유료 요금제(Pro, Business 등)별 월 제공 크레딧, 주요 기능 권한 표시
- **구독하기 버튼**:
  - 비로그인: 클릭 시 '인증 모달' 표시
  - 로그인: 클릭 시 결제 수단 등록 또는 결제 확인 화면(설정 모달)으로 이동

---

## 크레딧 충전 (Credit Recharge)

### 기능

- 구독과 별개로 필요한 만큼의 크레딧을 즉시 구매

### 동작

- **충전 옵션**: 단위별(예: 100, 500, 1000 크레딧) 가격 정보 카드 표시
- **구매하기 버튼**:
  - 비로그인: 클릭 시 '인증 모달' 표시
  - 로그인: 클릭 시 해당 금액 결제 프로세스 시작

---

## 요금 FAQ (하단)

### 기능

- 결제, 환불, 구독 해지 등 요금 관련 자주 묻는 질문 답변 제공

### 동작

- **아코디언(Accordion)**: 질문 클릭 시 하단으로 답변 내용이 펼쳐지거나 접힘
- **항목 예시**: 구독 취소 방법, 크레딧 이월 여부, 환불 규정 등

---

## 공통 동작

- **데이터 동기화**: 앱 내부 '설정 > 구독관리'에서 제공하는 요금제/크레딧 정보와 실시간으로 동일한 데이터를 표시
- **통화 표시**: 한국 사용자 대상이므로 원화(₩) 및 부가세 포함 여부 명시
