# 홈페이지 > 랜딩 페이지 기능명세서

## 페이지 개요

- **목적**: 서비스 소개 및 사용자 가입 유도(CTA)를 위한 랜딩 페이지
- **접근 권한**: 전체 (비로그인 사용자 대상)

---

## 히어로 섹션 (Hero Section)

### 기능

- 서비스의 핵심 가치(Value Proposition) 전달 및 즉각적인 가입 유도

### 동작

- **메인 카피**: 서비스의 목적을 명확히 전달하는 대형 타이틀
- **시작하기 버튼(CTA)**: 클릭 시 '인증 모달' 표시
- **비주얼 요소**: 서비스 결과물의 퀄리티를 보여주는 고품질 배경 영상

---

## 기능 소개 섹션 (Feature Section)

### 기능

- 주요 기능 설명 및 홍보 영상 노출

### 동작

- **기능 카드**: 제목, 짧은 설명 및 홍보 영상 표시

---

## 파트너/신뢰 섹션 (Partner Section)

### 기능

- 주요 고객사(광고대행사 등) 또는 사용 실적을 보여주어 신뢰도 확보

### 동작

- **로고 슬라이더**: 제휴사 또는 서비스 활용 기업들의 로고 나열

---

## 갤러리 프리뷰 섹션 (Gallery Preview Section)

### 기능

- 실제 생성된 고퀄리티 이미지/영상을 일부 노출하여 서비스 효능감 체감

### 동작

- **미리보기 그리드**: '공개 갤러리'의 인기 생성물 일부를 카드로 표시
- **더 보기 버튼**: 클릭 시 `홈페이지 > 공개 갤러리` 페이지로 이동

---

## 공통 동작

- **스크롤 애니메이션**: 섹션 진입 시 부드러운 페이드인 또는 슬라이드 효과 적용
