# 홈페이지 헤더 기능명세서

## 컴포넌트 개요

- **목적**: 서비스 전반의 주요 메뉴 이동 및 로그인/가입 접근점 제공
- **사용 위치**: 홈페이지 영역 상단 고정
- **접근 권한**: 전체 (로그인 여부에 따라 일부 버튼 변경)

---

## 네비게이션 메뉴

### 기능

- 서비스의 핵심 페이지로의 이동 링크 제공

### 동작

- **로고**: 클릭 시 `홈페이지 > 랜딩 페이지`로 이동
- **AI 스튜디오**: 클릭 시 `앱 > 홈`으로 이동 (로그인 필수)
- **갤러리**: 클릭 시 `홈페이지 > 공개 갤러리`로 이동
- **구독/가격**: 클릭 시 `홈페이지 > 구독/가격`으로 이동

---

## 사용자 액션 버튼

### 기능

- 사용자의 로그인 상태에 따른 인증 관련 기능 제공

### 동작

- **비로그인 상태**:
  - **로그인 버튼**: 클릭 시 '인증 모달(로그인)' 표시
  - **시작하기 버튼**: 클릭 시 '인증 모달(회원가입)' 표시 (강조된 스타일)
- **로그인 상태**:
  - **프로필**: '로그인' 및 '시작하기' 버튼 대신 사용자 프로필/이름 표시 및 '로그아웃' 버튼 표시

---

## 공통 동작

- **스크롤 반응**: 페이지 스크롤 시 배경색이 투명에서 불투명으로 변경 및 상단에 고정됨(Sticky)
- **반응형 대응**: 화면 크기가 작아질 경우 메뉴 항목이 햄버거 아이콘(드로워)으로 통합됨
