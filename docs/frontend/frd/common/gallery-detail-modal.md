# 생성물 상세보기 모달 기능명세서

## 컴포넌트 개요

- **목적**: 갤러리, 라이브러리, 생성 페이지 등에서 생성된 컨텐츠(이미지, 동영상, 문서)를 전체화면으로 상세 확인
- **사용 위치**:
  - `앱 > 홈` (갤러리 섹션)
  - `앱 > 라이브러리`
  - `앱 > 홍보 이미지`
  - `앱 > 영상 컨텐츠`
  - `홈페이지 > 공개 갤러리`
- **레이아웃**: 좌측 확대된 생성물 + 우측 상세 정보 및 액션

---

## 좌측 영역 (생성물 표시)

### 기능

- 확대된 이미지, 동영상 또는 문서(리포트/문구) 디스플레이
- 영상은 플레이어로 재생 가능 (음소거, 자동재생 안 함)
- 반응형 크기 조정

### 구성

- **생성물 미디어**: 이미지(전체화면 표시) / 동영상(플레이어로 재생) / 문서(뷰어 또는 미리보기)
- **좋아요 수**: 하단 좌측에 하트 아이콘 + 좋아요 개수 표시 (항상 표시)
- **좋아요 버튼**: 타인 생성물일 때만 하트 토글 버튼 표시 (로그인 필수)
- **만족도 피드백 버튼**: 내 생성물일 때만 “좋아요/싫어요(만족도)” 버튼 표시 (로그인 필수, 단 1회)
- **X 버튼**: 상단 우측 모달 닫기

### 상태별 처리

- **이미지 로딩 중**: 스켈레톤 또는 로딩 표시
- **영상 로딩 중**: 플레이어 로딩 표시
- **로드 실패**: 에러 메시지 표시

---

## 우측 영역 (정보 및 액션)

### 기능

- 생성물의 상세 정보와 관련 액션 버튼 제공
- 스크롤 가능한 영역 (정보가 많을 경우)

### 구성

#### 1) 만든이 정보 (상단)

- **아바타**: 만든이의 프로필 사진 (기본값: 기본 아이콘)
- **닉네임**: 만든이 이름

#### 2) 생성 정보 (중간)

- **브랜드명**
  - **설명**: 생성물에 사용된 브랜드 이름
  - **표시 조건**: 항상
- **생성 모델**
  - **설명**: 사용된 AI 모델명 (예: DALL-E, Midjourney)
  - **표시 조건**: 항상
- **품질**
  - **설명**: 생성 품질 설정값
  - **표시 조건**: 항상
- **크기**
  - **설명**: 가로 x 세로 (픽셀 단위, 예: 1920x1080)
  - **표시 조건**: 항상
- **생성일**
  - **설명**: ISO 형식 또는 상대 시간 (예: "3일 전")
  - **표시 조건**: 항상

#### 3) 액션 버튼 (하단)

- **이 템플릿으로 바로 만들기**
  - **동작**: 해당 템플릿을 선택한 상태로 생성 페이지(홍보 이미지 또는 영상 컨텐츠)로 이동
  - **활성화 조건**: 항상
- **즐겨찾기 추가**
  - **동작**: 즐겨찾기 토글 (별 아이콘으로 시각적 피드백)
  - **활성화 조건**: 항상
- **다운로드**
  - **동작**: 원본 파일 다운로드
  - **활성화 조건**: 항상
- **공개/비공개**
  - **동작**: 내 생성물의 공개 여부를 토글
  - **활성화 조건**: 내 생성물 + 이미지/동영상일 때만 표시

---

## 동작

### 열기

- **갤러리 > 카드 "자세히 보기" 클릭** → 해당 생성물 상세 모달 열림
- **라이브러리 > 카드 클릭** (선택 모드 미활성화 시) → 해당 생성물 상세 모달 열림
- **생성 페이지 > 작업 내역의 카드 클릭** → 해당 생성물 상세 모달 열림

### 닫기

- X 버튼 클릭 → 모달 닫기
- 배경 클릭 → 모달 닫기
- ESC 키 → 모달 닫기

### 페이지 이동

- "이 템플릿으로 바로 만들기" 버튼 클릭 시:
  - **이미지 생성물**: 홍보 이미지 페이지로 이동 (템플릿 자동 선택)
  - **영상 생성물**: 영상 컨텐츠 페이지로 이동 (템플릿 자동 선택)

---

## 공개 설정 (갤러리 노출)

### 정책

- **기본값**: 이미지/동영상 생성물은 기본 비공개
- **변경 위치**: 생성물 상세보기 모달에서만 변경 가능
- **표시 조건**: **내 생성물 + 이미지/동영상**일 때만 “공개/비공개” 토글(또는 버튼) 표시
- **갤러리 노출 범위**:
  - 공개 전환 시: `홈페이지 > 공개 갤러리` + `앱 > 홈(앱 내 갤러리)`에서 노출됨
  - 비공개 전환 시: 위 갤러리들에서 즉시 노출 해제됨

### 상태별 처리

- **공개 전환 중**: 토글/버튼 로딩 표시, 중복 클릭 방지
- **공개 전환 실패**: 에러 메시지 표시 + 원래 상태로 롤백

## 피드백 정책

### 1) 내 생성물(만족도)

- **의미**: “내가 만든 결과물이 만족스러운지”에 대한 피드백
- **방식**: **좋아요/싫어요 중 1개를 단 1회만 선택**
- **적용 범위**: 내 생성물 전체(리포트/문구/이미지/동영상)
- **UI**: 내 생성물일 때만 “좋아요/싫어요(만족도)” 버튼 노출
- **제약**: 1회 선택 후에는 변경/취소 불가(선택 결과만 표시)

### 2) 타인 생성물(좋아요)

- **의미**: 갤러리에서 타인 생성물에 대한 호감 표시
- **방식**: **좋아요/좋아요 취소 토글**
- **UI**: 타인 생성물일 때만 하트 토글 버튼 노출

---

## 상태별 처리

- **로딩 중**: 스켈레톤 UI 표시
- **로드 실패**: 에러 메시지 + 재시도 버튼 또는 닫기
- **비로그인 상태**: ‘바로 만들기’, ‘즐겨찾기’, ‘다운로드’, ‘좋아요(타인)’, ‘만족도(내)’ 클릭 시 로그인 모달 표시
- **내 생성물**: 만족도 피드백(1회), 공개/비공개(이미지/동영상), 즐겨찾기, 다운로드, 바로 만들기 가능
- **타인 공개 생성물**: 좋아요(토글), 즐겨찾기, 다운로드, 바로 만들기 가능
