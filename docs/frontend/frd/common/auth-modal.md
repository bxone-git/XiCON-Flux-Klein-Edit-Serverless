# 인증 모달 기능명세서

## 컴포넌트 개요

- **목적**: 소셜 로그인을 통한 사용자 인증 및 회원가입
- **사용 위치**: 홈페이지 (홈페이지 헤더의 로그인/시작하기 버튼 클릭 시, 앱 내부 접근 시, 앱 사용 중 인증 만료시 등)
- **인증 방식**: 소셜 로그인만 지원 (네이버, 카카오톡, 구글)
- **자체 로그인**: 없음

---

## 로그인

### 기능

- 기존 사용자의 소셜 계정으로 로그인
- 지원 소셜 로그인: 네이버, 카카오톡, 구글

### 동작

- 소셜 로그인 버튼 클릭 → 해당 소셜 서비스 인증 페이지로 전환
- 인증 완료 후 → `App > 홈`으로 이동

### 상태별 처리

- **로딩 중**: 로딩 표시
- **인증 실패**: 에러 메시지 표시
- **인증 취소**: 모달 유지

---

## 회원가입

### 기능

- 신규 사용자의 소셜 계정으로 회원가입
- 지원 소셜 로그인: 네이버, 카카오톡, 구글

### 동작

- 소셜 로그인 버튼 클릭 → 해당 소셜 서비스 인증 팝업 표시
- 인증 완료 후 → `앱 > 홈`으로 이동하며 온보딩 흐름 시작

### 상태별 처리

- **로딩 중**: 로딩 표시
- **인증 실패**: 에러 메시지 표시
- **인증 취소**: 팝업 닫기

---

## 온보딩 모달

### 기능

- 최초 가입 사용자를 위한 간단한 스테퍼
- 서비스 이용을 위한 최소한의 사용자 프로필 및 환경 설정

### 스테퍼 구성 (단계별 폼)

- **1단계**
  - **제목**: 프로필 설정
  - **주요 항목**: 닉네임 입력
  - **비고**: 소셜 계정 이름 기본값 제공
- **2단계**
  - **제목**: 이용 목적 선택
  - **주요 항목**: 사용 역할 (대행사, 브랜드 매니저, 개인 등)
  - **비고**: 객관식 단일 선택
- **3단계**
  - **제목**: 시작하기
  - **주요 항목**: 서비스 가이드 동영상 재생
  - **비고**: '건너뛰기' 또는 '시작하기' 버튼

### 동작

- **자동 시작**: 소셜 회원가입 완료 후 자동으로 `앱 > 홈`으로 이동하며 노출
- **단계 이동**: 각 단계의 필수 항목 입력 시 '다음' 버튼 활성화
- **최종 완료**: 가이드 영상 시청(또는 건너뛰기) 후 '시작하기' 클릭 시 `앱 > 홈`으로 이동하며 온보딩 완료

### 상태별 처리

- **표시 조건**: 최초 가입 사용자만 (온보딩 미완료 유저)
- **입력 중 이탈**: 다음 접속 시 완료하지 않은 단계부터 다시 시작
- **이미 완료**: 일반 로그인 시에는 표시 안 함 (온보딩 완료 시 표시 안 함)

## 모달 닫기

### 동작

- X 버튼 클릭 → 모달 닫기
- 배경 클릭 → 모달 닫기
- ESC 키 → 모달 닫기

### 상태별 처리

- **로그인/가입 진행 중**: 닫기 불가 또는 확인 다이얼로그 표시
